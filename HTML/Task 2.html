What is HTML ?

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating web pages. It descibes the structure of a webpage. HTML consists of a series of
elements.

What is CSS ?

CSS stands for Cascading Style Sheets. CSS descibes how HTML elements are to be displayed on screen, paper or other media. CSS saves a lot of work. It can control the layout of
multiple web pages all at once.

What is Javascript ?

Javascript is the world's most popular programming language of web development. It is easy to learn.

What is bootstrap ?

Bootstrap is a front end framework for faster and easier wed development. Bootstrap also gives you the ability to easily create responsive designs.
